## 🔧 الإعداد الاختياري (للإنتاج فقط)

### متى تحتاج إلى إعداد متغيرات البيئة؟

**فقط إذا كنت ستنشر الموقع على الإنترنت (Production)**

### الخطوات الاختيارية:

#### 1. إنشاء ملف .env
```bash
# نسخ ملف المثال
cp .env.example .env
```

#### 2. اختيار خيار التخزين

**الخيار 1: Upstash Redis (موصى به للإنتاج)**
1. اذهب إلى: https://upstash.com/
2. أنشئ حساب مجاني
3. أنشئ قاعدة بيانات Redis
4. انسخ URL و Token
5. أضفها إلى ملف .env:
```env
UPSTASH_REDIS_REST_URL=https://your-url.upstash.io
UPSTASH_REDIS_REST_TOKEN=your_token
```

**الخيار 2: Vercel KV (بديل)**
1. اذهب إلى: https://vercel.com/storage/kv
2. أنشئ قاعدة بيانات KV
3. انسخ URL و Token
4. أضفها إلى ملف .env:
```env
KV_REST_API_URL=https://your-url.kv.vercel-storage.com
KV_REST_API_TOKEN=your_token
```

**الخيار 3: ملف محلي (افتراضي)**
- لا يحتاج أي إعداد
- يعمل تلقائياً للتطوير المحلي

#### 3. إضافة مفتاح التشفير (اختياري)

```bash
# إنشاء مفتاح تشفير آمن
node -e "console.log(require('crypto').randomBytes(32).toString('hex'))"
```

أضف الناتج إلى ملف .env:
```env
MESSAGE_ENCRYPTION_KEY=your_generated_key_here
```

## 🧪 اختبار النظام

### اختبار إرسال رسالة:

1. افتح المتصفح على: `http://localhost:5173/contact`
2. املأ نموذج الاتصال
3. اضغط "إرسال"
4. ستحصل على رسالة نجاح ✅

### اختبار عرض الرسائل:

1. افتح: `http://localhost:5173/dashboard`
2. أدخل كلمة المرور: `00000008`
3. انتقل إلى قسم "Messages"
4. سترى الرسائل المرسلة ✅
